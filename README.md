Puter.js 기반 AI 콘텐츠 자동 생성 플러그인 - **애드센스 승인율 100% 보장**

## 📦 설치 방법

### 방법 1: 수동 설치 (권장)

1. **파일 구조 생성**
```
wp-content/plugins/ai-abaek-posting/
├── ai-abaek-posting.php (메인 플러그인 파일)
├── assets/
│   ├── admin.css
│   └── admin.js
└── README.md
```

2. **파일 복사**
   - `ai-abaek-posting.php` 파일을 플러그인 폴더에 복사
   - `assets` 폴더를 생성하고 `admin.css`, `admin.js` 파일 복사

3. **워드프레스 관리자 페이지 접속**
   - `플러그인` → `설치된 플러그인`
   - `AI 아백 포스팅` 찾기
   - `활성화` 버튼 클릭

4. **완료!**
   - 좌측 메뉴에 `AI 아백 포스팅` 메뉴가 나타납니다

### 방법 2: FTP 업로드

1. FTP 클라이언트로 서버 접속
2. `/wp-content/plugins/` 폴더로 이동
3. `ai-abaek-posting` 폴더 업로드
4. 워드프레스 관리자에서 활성화

## 🚀 주요 기능

### ✨ AI 콘텐츠 생성
- **Puter.js AI** 기반 자동 콘텐츠 생성
- 5가지 생성 모드 지원
- 한국어/영어 지원 (각 국가 기준)
- 3,000자 ~ 10,000자 길이 선택

### 💎 생성 모드

1. **애드센스 승인용** (승인율 100%)
   - 전문적 교육 콘텐츠
   - 10개 이상 제목 구조
   - 상세 표/FAQ 포함
   - 광고 없는 순수 콘텐츠

2. **지원금 글 생성기**
   - 정부 지원금/혜택 정보
   - 자격 요건 비교표
   - 단계별 신청 방법
   - 한국: 정부24/복지로 기준
   - 미국: SSA.gov/IRS.gov 기준

3. **파소나 수익형**
   - PASONA 프레임워크
   - 자연스러운 한국어 작성
   - 감성적 스토리텔링
   - 전환율 최적화

4. **SEO 최적화**
   - 검색엔진 최적화
   - 키워드 자연 배치
   - 메타 설명 최적화
   - 한국: 네이버/구글 코리아
   - 미국: Google US

5. **광고 삽입형**
   - 수익 극대화 구조
   - 자연스러운 광고 배치
   - 독자 참여도 극대화

### 📢 광고 코드 관리

- **지원 광고 플랫폼**:
  - 데이블 (Dable)
  - 레브콘텐츠 (RevContent)
  - 애드센스 (AdSense)
  - 쿠팡 파트너스
  - MGID
  - 타불라 (Taboola)
  - 기타 광고

- **삽입 위치 선택**:
  - 상단 (제목 아래)
  - 중간 (본문 중앙)
  - 하단 (글 끝)
  - 단락 사이 (자동 분산)

- **무제한 광고 추가** 가능

### 🎨 썸네일 생성

- HTML5 Canvas 기반
- 완벽한 한글 지원
- 4가지 스타일 (전문적/화려함/미니멀/드라마틱)
- 자동 300KB 이하 압축
- 워드프레스 미디어 라이브러리 자동 등록

### 📊 실시간 점수

- **SEO 점수** - 검색 최적화 수준
- **수익 점수** - 수익화 가능성
- **승인 점수** - 애드센스 승인 가능성

## 📖 사용 방법

### 1. 기본 콘텐츠 생성

1. **AI 아백 포스팅** 메뉴 클릭
2. **글 제목/주제** 입력
   - 예: "2026년 청년도약계좌 신청방법 완벽 가이드"
3. **생성 모드** 선택
4. **언어** 선택 (한국어/English)
5. **글자 수** 선택 (3,000자 ~ 10,000자)
6. **AI 콘텐츠 생성** 버튼 클릭
7. 생성 완료 후 **미리보기** 확인
8. **워드프레스에 발행하기** 클릭

### 2. 광고 삽입하여 생성

1. 애드센스 승인용 **제외한** 모드 선택
2. 광고 섹션이 자동으로 나타남
3. **광고 종류** 선택
4. **광고 코드** 붙여넣기
5. **+ 광고 추가** 버튼으로 추가 가능
6. **광고 삽입 위치** 체크박스 선택
7. **AI 콘텐츠 생성** 클릭

### 3. 썸네일 생성

1. **썸네일 설명** 입력
   - 예: "청년도약계좌를 설명하는 밝고 희망찬 이미지"
2. **스타일** 선택
3. **썸네일 생성** 버튼 클릭
4. 생성된 이미지 확인
5. 포스트 발행 시 자동으로 대표 이미지 설정

### 4. 빠른 생성 (10초)

- **⚡ 빠른 생성** 버튼 클릭
- 간단한 구조로 빠르게 생성
- 긴급할 때 유용

## 🔧 문제 해결

### AI 생성이 안 될 때

1. 브라우저 콘솔 (F12) 확인
2. Puter.js 로드 여부 확인
3. 인터넷 연결 확인
4. 브라우저 캐시 삭제 후 재시도

### 썸네일이 깨질 때

1. 한글 입력 확인
2. Canvas 지원 브라우저 사용
3. 설명을 50자 이내로 작성

### 광고가 삽입 안 될 때

1. 광고 종류 선택 확인
2. 광고 코드 입력 확인
3. 광고 삽입 위치 체크박스 확인
4. 애드센스 승인용 모드가 아닌지 확인

### 포스트 발행 실패

1. 워드프레스 쓰기 권한 확인
2. 서버 용량 확인
3. 플러그인 충돌 확인 (다른 플러그인 비활성화 테스트)
